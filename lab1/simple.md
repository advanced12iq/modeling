# simple.md

## О чем эта работа
Мы сравниваем два способа посчитать полет шарика:
1. **Без воздуха** (модель Галилея).
2. **С воздухом** (модель Ньютона + сопротивление).

Условия одинаковые:
- угол `45°`
- скорость старта `1 м/с`
- старт из точки `(0, 0)`

## Идея модели Галилея
Если воздуха нет, по оси `x` скорость постоянная, а по `y` есть только ускорение вниз `g`.
Получается красивая парабола.

Из формул сразу считаем:
- сколько летит тело,
- куда упадет,
- максимальную высоту.

## Идея модели Ньютона с воздухом
Тут есть сопротивление воздуха. Оно тем сильнее, чем больше скорость.
Поэтому мы вводим две скорости:
- `u` — по горизонтали,
- `w` — по вертикали.

Система уравнений:
- `dx/dt = u`
- `dy/dt = w`
- `du/dt = -(beta/m) * u * sqrt(u^2 + w^2)`
- `dw/dt = -g -(beta/m) * w * sqrt(u^2 + w^2)`

Коэффициент:
- `beta = c * S * rho / 2`
- `c=0.15`, `S=3`, `rho=1.225`.

## Почему тут нужен Рунге-Кутта
Для этой системы нет удобного простого решения "в одну формулу", поэтому считаем по маленьким шагам времени.
Используем метод Рунге-Кутты 4-го порядка (RK4): он точнее обычного Эйлера и стабильно работает.

## Как нашли точку падения точнее
Компьютер считает по шагам, и момент `y=0` обычно попадает между двумя шагами.
Поэтому берем две соседние точки и делаем линейную интерполяцию. Так `x` в точке падения получается точнее.

## Что получилось
Для базового случая:
- **Галилей**: `x_land = 0.101937 м`
- **Ньютон + воздух**: `x_land = 0.061637 м`

С сопротивлением тело падает ближе.

## Почему это логично
Воздух тормозит движение:
- горизонтальная скорость убывает,
- вверх тело поднимается меньше,
- вниз начинает падать раньше.

Поэтому дальность и высота меньше, чем в идеальной модели без воздуха.
