# simple.md

## О чем работа
Сравниваются два способа расчета полета шарика:
1. Модель Галилея (без сопротивления воздуха).
2. Модель Ньютона (с сопротивлением воздуха, решение через RK4).

Новые данные задачи:
- материал: **медь**;
- угол: `57°`;
- начальная скорость: `160 м/с`;
- радиус: `0.25 м`;
- старт: `(0, 0)`.

## Модель Галилея
Если воздух не мешает, то движение описывается простыми формулами.
Траектория получается параболой.
По формулам сразу находим:
- время полета,
- дальность,
- максимальную высоту.

## Модель Ньютона с воздухом
Когда учитываем воздух, появляется сила сопротивления.
Она направлена против скорости и растет при увеличении скорости.

Используются две компоненты скорости:
- `u` по оси `x`;
- `w` по оси `y`.

Коэффициент сопротивления:
- `beta = c * S * rho_air / 2`,
- где `c=0.15`, `S=3.0`, `rho_air=1.225`.

## Почему нужна масса и как она считается
Радиус задан напрямую: `r = 0.25 м`.
Материал теперь медь, поэтому берем плотность меди:
- `rho_copper = 8960 кг/м^3`.

Масса шара:
- `m = rho_copper * 4/3 * pi * r^3`.

В этой работе получилось `m = 586.430629 кг`.

## Почему применили Рунге-Кутту
Для модели Ньютона удобной готовой формулы для всей траектории нет.
Поэтому считаем движение шаг за шагом методом Рунге-Кутты 4-го порядка.
Он точный и устойчивый для такой задачи.

## Как нашли точку падения точно
Компьютер считает по времени с шагом `dt`, поэтому момент `y=0` обычно между двумя шагами.
Чтобы не терять точность, берем эти две точки и делаем линейную интерполяцию.
Так получаем точный `x_land`.

## Итоговые числа
Для новых входных данных:
- Галилей: `x_land = 2383.971836 м`, `t_land = 27.357246 с`, `y_max = 917.747757 м`.
- Ньютон: `x_land = 1253.623389 м`, `t_land = 22.331246 с`, `y_max = 614.885546 м`.

Разница дальности:
- `delta_x = -1130.348448 м`.

## Главный вывод
С сопротивлением воздуха шарик летит заметно меньше и ниже.
При большой начальной скорости (`160 м/с`) отличие от идеальной модели без воздуха очень большое.
